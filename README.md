# Mobile System

This is a unified repository for the Mobile System project.

## Project Structure
- **/Mobile-System-Backend**: Node.js/Express backend (Deploy to Render)
- **/Mobile-System-Front-End**: Vite/React frontend (Deploy to Netlify)

## Deployment Settings
### Backend (Render)
- Root Directory: `Mobile-System-Backend`
- Build Command: `npm install`
- Start Command: `npm start`

### Frontend (Netlify)
- Base Directory: `Mobile-System-Front-End`
- Build Command: `npm run build`
- Publish Directory: `dist`

## Environment Variables
Ensure all required environment variables are set in the respective hosting platforms as discussed.
